# BrainFuck
The BrainF*ck language interpreter and variatinos.  

# 概要
いわゆるBrainF*ck言語を標準入力またはファイルから読み込み、実行します。  

言語についてはWikipedia等を参照ください。  
Brainfuck - Wikipedia https://ja.wikipedia.org/wiki/Brainfuck  

# 実装について
## コマンドライン引数
コマンドライン引数をつけずに実行すると、コードを標準入力から読み込みます。  
ファイルに記述したコードを実行するには -i オプションを使用し、"BrainFuck.exe -i ファイル名"とします。  

## 派生言語
通常のBrainF*ck処理系で使用する命令語のかわりに、任意の文字列を命令語とするプロジェクトを作成できます。  
ソリューション内のPontaプロジェクトを参照してください。  

## エラーハンドリング
最低限のエラーチェックとして、"["命令と"]"命令は一対一に対応していないと実行時エラーが発生します。  

## 命令語でない文字列
命令語に含まれない文字列はすべて無視して実行されます。  
したがって、任意のコメントを入れたり、読みやすいように改行を入れたりできます。  

# サンプル
Pontaプロジェクト内にサンプルファイルを用意しています。  

## 標準出力
"Ponta.exe -i nho.bf"とコマンドラインから実行することで、"んほー！！"と出力されます。  
実行環境としてはWindowsのコマンドプロンプトを想定しています。  

## 標準入力＆標準出力
"Ponta.exe -i echo.bf"とコマンドラインから実行することで、  
標準入力から１文字読み込み、そのまま標準出力に出力します。  

# ソリューション構成
├─BrainFuck  BrainF*ck言語の処理系です。  
│  
├─BrainFuckCore  言語処理系の本体です。各処理系の.exeファイルと同じフォルダにcore.dllを置いて実行します。  
│  
└─Ponta  BrainF*ck言語の命令語のかわりに、バファローズポンタの叫び声を命令語として実行する処理系です。
